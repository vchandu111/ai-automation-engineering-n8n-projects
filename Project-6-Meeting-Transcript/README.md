# **Project 6: Meeting Transcript to Structured Notes Automation**

## **Objective**

Design and build an **n8n workflow** that converts raw meeting transcripts into well-structured meeting notes using AI. The workflow should accept a transcript via a form or webhook, process it through a language model, save the generated notes to Google Docs, and email the summary to participants.

---

## **Learning Outcomes**

By completing this project, you should be able to:

* Accept long-form text input through a Form Trigger or Webhook.
* Use the Basic LLM Chain node to process and summarize content.
* Save AI-generated structured content to Google Docs.
* Send formatted email summaries using Gmail.
* Design an AI-assisted document automation workflow in n8n.

## **Project Requirements**

You are required to build an automation workflow that processes meeting transcripts and generates structured notes automatically.

### **Input Form / Webhook Fields**

Your form or webhook must accept the following information:

* Meeting Title
* Participant Emails (comma-separated)
* Raw Meeting Transcript (long text field)

---

## **Workflow Requirements**

The workflow must perform the following actions in sequence:

1. **Trigger** when a transcript is submitted via **Form Trigger** or **Webhook**.
2. **Send the transcript** to the **Basic LLM Chain** node for AI processing.
3. The AI must generate the following structured sections:
   * **Meeting Summary**
   * **Key Discussion Points**
   * **Action Items**
   * **Next Steps**
4. **Save the structured notes** to a new or existing **Google Doc**.
5. **Email the meeting summary** to all participants using **Gmail**.

---

## **Google Docs Requirements**

The Google Doc created or updated by the workflow must contain the following clearly labeled sections:

* Meeting Title
* Meeting Summary
* Key Discussion Points
* Action Items
* Next Steps

The content of each section should be populated using the AI-generated output.

---

## **Email Requirements**

The email sent to participants must:

* Include the meeting summary generated by the AI.
* Highlight the key action items from the meeting.
* Thank participants for attending.
* End with a friendly and professional closing message.

---

## **AI Prompt Guidelines**

When configuring the **Basic LLM Chain** node, your prompt should instruct the AI to:

* Read the provided meeting transcript.
* Write a concise meeting summary (3–5 sentences).
* List the key discussion points as bullet points.
* Extract all action items and assign them clearly.
* Suggest logical next steps based on the discussion.

You are responsible for writing a clear, effective prompt that produces well-structured output suitable for saving in Google Docs and sending via email.

---

## **Workflow Constraints**

* The workflow must begin with a **Form Trigger** or **Webhook** node.
* Use the **Basic LLM Chain** node to process the transcript.
* Use **Google Docs** to save the structured meeting notes.
* Use **Gmail** to send the summary email to participants.
* Ensure the AI output is correctly parsed and inserted into Google Docs and the email body.
* The workflow should execute automatically upon each transcript submission.

---

## **Expected Workflow**

The completed workflow should follow this sequence:

```
Form Trigger / Webhook
      ↓
Send Transcript to Basic LLM Chain (AI)
      ↓
Generate Summary, Discussion Points, Action Items & Next Steps
      ↓
Save Structured Notes to Google Docs
      ↓
Email Summary to Participants (Gmail)
```
